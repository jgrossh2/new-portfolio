SANTA'S HELPER

This is an application that was created to help with preparing for the holidays.

  • Users can create an account
  • Users can create, edit, and delete food recipes, drinks, and wishlists

TALLY MAX WORD GENERATOR

This is an application created especially for those who like to play word games.

  • User can search for words using up to 5 letters
  • User can click and drag letters from the keyboard into the dropzone
  • User has the choice of returning 2 or 3 letter words
  • Along with the words, the definition of the word, the pronunciation, and an image of the word is available.
  • Uses Webster Dictionary, Pexels, and Pixabay APIs

Group work. Technologies used: HTML, CSS, Javascript, JQuery, Materialize

TECH-BLOG

Tech blog was created as an online environment for users to write posts about the tech world.

  • Users can create an account.
  • Users can create, edit, delete, or comment on posts when logged in.
  • The home page shows all posts with comments
  • User dashboard shows user posts with edit functionality
  • Bcrypt utilized for password protection
  • Automatic sign out after 1 min of idle action

Technologies used: Javascript, Node, MYSQL2, Sequelize, express-handlebars, Bootstrap

WORK-DAY SCHEDULER

Work Day Scheduler was created to keep track of daily tasks.

  • User can create tasks for time slot during the day
  • During the current hour the row is shaded green
  • During a past hour the row is shaded gray
  • For future hours the row is shaded blue

Technologies used: HTML, CSS, Javascript, JQuery
